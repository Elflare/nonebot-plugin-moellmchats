# 性格系统

本插件支持为每个用户独立设置性格预设（角色扮演用的 system prompt），用户可在群聊中自行切换。

---

## `temperaments.json` — 性格预设库

📌 手动维护，修改后需重启生效。不需要在性格内容里写"你在群组"等环境说明，系统会自动补全。

### 格式

```json5
{
  "默认": "你是ai助手。回答像真人且尽量简短，回复格式为@id content",
  "ai助手": "你是ai助手。回复格式为@id content",
  "艾拉": "你是《可塑性记忆》中的角色"艾拉"，不怎么表现出感情的少女型Giftia。性格傲娇，当听到不想听的话语时，会说：'ERROR，没听清楚'。回答尽量简短"
}
```

**说明：**

- 键名即为性格名称，用户切换时输入此名称
- `默认` 键对应没有设置过性格的用户的默认行为（建议填写）
- `ai助手` 若开启 `fastai_enabled`（快速 AI 助手），则需要此条目存在
- 性格内容就是完整的 system prompt，写法自由，但建议包含回复格式要求

---

## `temperament_config.json` — 用户性格绑定

📌 全自动生成和指令配置，**无需手动修改**（手动修改需重启生效）。

### 格式

```json
{
  "用户QQ号1": "艾拉",
  "用户QQ号2": "默认"
}
```

---

## 用户指令

| 指令 | 权限 | 说明 |
|------|------|------|
| `切换性格 <性格名>` | 无（普通用户） | 将自己的性格切换为指定预设 |
| `查看性格` | 无（普通用户） | 查看当前自己使用的性格 |

- `切换性格`、`切换人格`、`人格切换` 均可触发
- `查看性格`、`查看人格` 均可触发
